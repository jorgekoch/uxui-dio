🎯 Objetivo do Projeto

O site do Wertmore tem como objetivo principal:

Converter visitantes em contatos qualificados (WhatsApp/email) através de uma experiência visual forte, clara e confiável.

🧠 Contexto do Usuário
👤 Perfil principal
Desenvolvedor de jogos (indie ou pequeno estúdio)
Conhecimento técnico variável
Nem sempre sabe avaliar qualidade de pixel art profundamente
⚠️ Problema central

O usuário precisa contratar alguém, mas:

Não sabe comparar artistas com precisão
Tem medo de escolher errado
Decide com base em percepção visual + confiança
💡 Estratégia do Site

O site não deve apenas “mostrar trabalhos”.

Ele deve:

Gerar impacto imediato
Facilitar julgamento visual
Reduzir incerteza
Guiar até o contato
🧭 Jornada do Usuário (Base do Wireframe)
1. Descoberta

Usuário chega via:

Google
indicação
Estado mental:
muitas opções
pouca confiança
2. Primeiro impacto (CRÍTICO)
Pergunta do usuário:

“Isso é bom mesmo?”

Micro decisão:

“fico ou saio?”

O que resolve:
Hero visual forte
Estética diferenciada
Clareza imediata
3. Julgamento visual
Pergunta:

“Isso serve pro meu jogo?”

Comportamento:
escaneia rapidamente
compara com outros artistas
O que resolve:
artes grandes
variedade controlada
consistência visual
4. Validação
Pergunta:

“Esse cara é profissional ou só artista?”

O que resolve:
serviços bem explicados
visão de projeto (não só arte)
posicionamento claro
5. Decisão
Pergunta:

“Vale a pena falar com ele?”

Risco percebido:
medo de escolher errado
O que resolve:
CTA claro
facilidade de contato
comunicação simples

🧠 Princípios de UX
1. Clareza > criatividade

Se o usuário não entende, falhou.

2. Impacto nos primeiros 3 segundos

Hero define retenção.

3. Mostrar > explicar

Arte primeiro, texto depois.

4. Reduzir incerteza

Cada seção deve responder uma dúvida.

5. Caminho simples

Sempre levar ao contato.

🚀 Critérios de Qualidade

Antes de desenvolver:

 O hero prende atenção imediatamente?
 Dá pra entender o valor sem ler muito?
 As artes são protagonistas?
 Existe diferenciação visual?
 O caminho até o contato está claro?
🔚 Resumo Estratégico

O site deve fazer o usuário sentir:

“Esse cara entende de jogo, tem estilo e provavelmente vai resolver meu problema.”
